Commonly Used Elements

div

Divs are purely structural elements, generally used for formatting information that is stored in other elements.

p

Paragraphs are used exactly the way you'd expect, to separate paragraphs of text in a document. They of course can be used for anything else as well, but their default behavior suggests that they are to be used as paragraph markers.

It's important to note that all of these elements are fully customizable, and most of them don't mean anything except for what their default formatting is. With some quick CSS, you can change all h1 elements into paragraphs and vice versa.

a

a stands for "anchor", but really this is just a hyperlink.
The href property decides where it leads.

br /

br / makes a new line in the document

The / indicates that this is a "self closing tag," meaning that it does not ever have a body and therefore doesn't need a closing tag
(<br /> is the same thing as <br></br>).

whatever-you-want

Though it is not recommended to do this, css is able to find an element with any name and define its style.

CSS Selector Examples

*

Selects every single element

div

Select all div elements

p, h1

Select all paragraph elements and all h1 elements

.class

Select all elements with class name "class"

#id

Select a SINGLE element with the id "id"

div > h1

Select every h1 element whose parent is a div

*:first-child

Select all elements who are the first child of their parent

div.element > h2:first-child

Select all h2 elements who are the first child of a div with class "element"

div::before

Select the space before every div element
